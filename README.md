# Every Camping
- 캠핑 용품 쇼핑몰
- 시연 영상: [https://www.youtube.com/watch?v=nZXMN6WChhE](https://www.youtube.com/watch?v=nZXMN6WChhE)

## 기술 스택
React, TypeScript, React Router, React Query, Vite, Axios, Tailwind CSS, daisyUI

## 프로젝트 기능
- 회원가입(이메일 인증), 로그인, 카카오 로그인
- 상품 CRUD, 리뷰 CRUD
- 1:1 채팅
- 상품 검색
- 장바구니, 주문조회, 일일 정산 조회
- 주문 확정 및 취소
- 판매자 승인 요청 관리

## 프로젝트 설명
- 캠핑 용품 쇼핑몰
- 반응형으로 제작
- 요즘 제일 많이 사용하는 라이브러리로 많은 커뮤니티가 형성되어 있어 자료를 쉽게 찾을 수 있고 UI를 보다 쉽게 구현하기 위해 React로 구현을 했습니다.
- SPA으로 구현되어 새로고침 없이 화면이 전환되어 보다 높은 UX를 제공합니다.
- React에서 제공하는 Virture DOM 기능으로 DOM 업데이트 시 실제로 변경되는 부분만 업데이트해주기 때문에 브라우저 렌더링 성능 개선을 기대할 수 있습니다.
- TypeScript를 사용하여 Type 체크를 받음으로써 JavaScript 보다 좀 더 안전하고 명확하게 작업을 진행했습니다.
- React Router를 사용해 Routing이 가능하도록 만들어주었습니다.
- React Query를 사용해 Redux 보다 좀 더 쉽고 간편하게 Sever State를 관리하였습니다.
- 통일된 스타일과 좀 더 간편하고 빠르게 스타일링하기 위해 Tailwind CSS와 daisyUI를 사용했습니다.
